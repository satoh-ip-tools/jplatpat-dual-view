# プライバシーポリシー / Privacy Policy

## 2列表示ビューア for J-PlatPat(非公式)

最終更新日:2026年7月16日

### 収集する情報

本拡張機能は、利用者の個人情報、閲覧履歴、検索内容、ページ内容、その他いかなるデータも収集しません。

### 外部への送信

本拡張機能は通信機能を持たず、いかなるデータも外部のサーバーへ送信しません。

### 保存する情報

本拡張機能の設定値(2列表示のON/OFF、図面ペインの幅、位置記憶のON/OFF、ハイライト語)は、ブラウザの拡張機能用ストレージ(`chrome.storage.local`)に保存されます。これらは利用者のブラウザ内にのみ保存され、開発者を含む第三者がアクセスすることはありません。

### 動作範囲

本拡張機能は `https://www.j-platpat.inpit.go.jp/` 配下のページでのみ動作し、表示レイアウトの変更のみを行います。J-PlatPatのサーバーに対して、通常のブラウザ閲覧以外のリクエストを送信することはありません。

### 免責

本拡張機能は個人が開発した非公式ツールであり、独立行政法人工業所有権情報・研修館(INPIT)および特許庁とは一切関係がありません。

### English Summary

This extension does not collect, store, or transmit any personal data, browsing history, or page content. It has no network capability. Settings are stored only in the browser's local extension storage (`chrome.storage.local`). It operates only on `https://www.j-platpat.inpit.go.jp/` and only modifies page layout. This is an unofficial tool and is not affiliated with INPIT or the Japan Patent Office.
